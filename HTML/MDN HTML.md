##HTML reference on MDN
####Block-level elements

`<div>`
Document division

`<footer>`
Section of page footer

`<form>`
Input form

`<h1>` through `<h6>`
Heading levels 1-6

`<header>`
Section or page header

`<p>`
Paragraph

`<section>`
Section of a web page

`<ul>`
Unordered list

`<table>`
Table

`<article>`
Article content
